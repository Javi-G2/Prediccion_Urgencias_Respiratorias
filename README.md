# Prediccion_Urgencias_Respiratorias
Este repositorio contiene el código desarrollado para la tesis de Magíster de Ciencia de Datos enfocada en la anticipación de la demanda semanal de urgencias respiratorias en 10 regiones de Chile, integrando variables sanitarias y ambientales.

El objetivo principal fue evaluar la viabilidad técnica de un enfoque predictivo multivariable. Se implementaron y compararon cuatro arquitecturas, Holt-Winters, Prophet, SARIMAX y Redes Neuronales Recurrentes.
